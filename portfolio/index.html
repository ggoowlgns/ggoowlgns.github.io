  • Edge Computing을 활용한 무인 마켓 : HUFS GO

    2018.12 ~ 2019.07
    • 무인 마켓 시장의 선두 주자인 Amazon Go를 벤치마킹하고 Edge Computing Platform을 도입한 프로젝트이다. 여러개의 Terminal Device 에서 촬영된 정보와 센싱 데이터들을 전부 Edge Server로 전송하고 Edge Server에서는 각종 데이터들을 조합하여 Object Detection, Face Racognition, QR-Scanner, Data Fusion 등의 작업을 실행한다. Cloud Server에서는 여러 Edge Server로 부터 고객정보, 구매 정보를 수집하여 데이터 분석(러닝-학습)하고 결과물을 다시 모든 Edge Server로 전송해준다.
      Edge Computing platform의 도입으로 1)사용자 응답 시간 감소, 2)고객 사생활 보호, 3)Masking cloud outage의 실현을 가능하게 하였다.

  • VR를 이용한 실시간 스트리밍 원격제어 차량 : VICER

    2018.03 ~ 2018.11
    • 대리운전기사들의 편의를 도모하고, 안정성을 확보하기 위해서 app device를 대리운전기사에게 제공한다. 운전기사는 컨트롤러를 사용하여 실내에서 원격으로 차량을 제어한다. 또한 운전석에서 보는 것과 같은 시야를 VR 어플리케이션을 통하여 제공받는다. 이를 통하여 실제 운전하는 것과 같은 효과를 얻을 수 있다. 360도 카메라는 삼성 "Gear360"를, VR 장비와 이에 부착되는 핸드폰으로는 "삼성 기어 VR"과 "삼성 갤럭시 8" 를 사용하였다.
      RTMP 를 참조하여 응용 프로토콜(TCP based)을 자체 개발하였다. 사용자는 VR 장비에 부착된 App을 통해 360도 영상을 실시간으로 제공받는다. 실제 운전이 가능하도록 핸들, 페달을 자체 제작하였고 영상정보를 통해 차량을 제어한다.
